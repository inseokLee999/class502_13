필요한 데이터만 쏙 출력하는 WHERE절
- 조건절
1. 문법 
SELECT ... FROM 테이블 명 WHERE 조건식;

참고) 조건식 - 참거짓 판별식 - 주로 많이 사용하는 연산 : 비교연산, 논리연산

2. 실습 

여러 개 조건식을 사용하는 AND, OR 연산자 :  다른 연산자를 함께 여러개 사용하는 경우
1. AND 연산자
	: 모든 조건이 참일 때 참
2. OR 연산자 
	: 어떤 조건이든 하나만 참이면 참인 조건

연산자 종류와 활용 방법 알아보기
1. 산술 연산자 
	+,-,*,/ -> 숫자 : 수치 연산 
	- 값이 있을 때만 연산 가능, NULL 상태는 연산 불가
	- nvl (컬럼명, NULL일때 대체 값)
	
	참고 ) DUAL : 가상 테이블, 값을 연산시 사용
	NULL : 값이 없음
	
2. 비교 연산자
	<,<=,>,=>
3. 등가 비교 연산자
	= : 같다
	같지않다
	!=
	<>
	^=
4. 논리 부정 연산자
	NOT 연산자 : 참인 조건 -> 거짓인 조건
5. IN 연산자
	-OR 연산자를 대체
	-DEPTNO IN (10,20,30) -> DEPTNO가 10,20,30 중에 포함되어있으면 참
6. BETWEEN A AND B 연산자
A 이상 B 이하
7. LIKE 연산자와 와일드 카드
	1) 문법
		컬럼명 LIKE '키워드' -> 컬럼명과 키워드가 일치하는 경우
		컬럼명 LIKE '키워드%' -> 키워드로 시작하는 패턴
		컬럼명 LIKE '%키워드' -> 키워드로 끝나는 패턴
		컬럼명 LIKE '%키워드%' -> 키워드가 포함된 패턴
		SELECT  * FROM EMP_LIKE WHERE ENAME LIKE '_*%%' ESCAPE '*';//*가 탈출문자
	2) -, %
	
	_:글자 1개 ENAME LIKE '_L%' 두번째 글자가 L인경우
	3) 와일드 카드 문자가 데이터의 일부일 경우 - ESCAPE 절 사용 
		- _,% 를 문자 그대로 패턴으로 인식하고 싶은경우

8. IS NULL 연산자
	NULL은 값이 없는 상태 : 연산이 X, 산술 비교 논리...
	
	컬럼명 IS NULL : 컬럼이 NULL 인상태
	
	참고) 컬럼명 IS NOT NULL : 컬럼이 NULL 이 아닌 상태
	
		오라클 : '',NULL은 동일
		MySQL : '' - 빈문자열, NULL - 값이 없음
				(컬럼명 <> "AND 컬럼명 IS NOT NULL
9. 집합 연산자
	SQL: 관계 대수 + 집합
1) UNION : 합집합
	- 중복 X
2) UNION ALL : 합집합
	- 중복 O
3) MINUS : 차집합
4) INTERSECT : 교집합
