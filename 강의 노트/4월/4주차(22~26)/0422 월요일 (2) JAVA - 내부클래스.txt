내부 클래스
- 클래스 안에 정의된 클래스 

1. 인스턴스 내부 클래스 
- 인스턴스 자원 비슷한 성격 
(멤버 변수, 인스턴스 메서드)
- 외부 클래스가 객체가 되어야만 객체를 생성할 수 있는 클래스  
-> 객체 생성과 상관 없는 자원(정적 변수, 정적 메서드는 정의 X - JDK15)
-> JDK16+, 정적 자원 정의 가능


2. 정적 내부 클래스 
  (static) 
  - 내부 클래스에 static이 추가 된 형태 
  - 정적 자원과 비슷한 성격 
  (정적 변수, 정적 메서드)
  - 외부 클래스가 객체가 되는 조건과 상관 X
  - 객체 생성과 상관없이 접근가능하다 (외부 클래스명으로 접근 가능하다)
  - 일반 클래스와 차이가 많이 없다.
  - 외부 클래스의 인스턴스 자원 접근 X 	
	
3. 지역 내부 클래스  
- 메서드 내부에 정의된 클래스 

인터페이스,추상클래스가 객체가 되는 조건
1)지역 내부에서 객체 생성
2) 미 구현된 메서드의 재정의를 통해서
