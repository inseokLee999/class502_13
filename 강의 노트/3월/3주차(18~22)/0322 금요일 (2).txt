4. 스타일 정의 방법
	1) 내부 스타일 시트
		- 웹 문서 안에서 사용할 스타일을 문서 안에 정리한 것
		<style>
			
		</style>
	2) 외부 스타일 시트
		- <link rel="stylesheet" type="text/css" href="외부스타일파일경로">
		-확장자 : 파일명.css
		
	3) 인라인 스타일 
		태그 속성 - style 속성에 직접 스타일 정의
