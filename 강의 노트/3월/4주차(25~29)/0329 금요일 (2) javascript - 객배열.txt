배열
1. 배열의 기초
-Array 생성자로 만들어진 객체
-데이터 군
-여러 데이터
-참고) 자바)
	컬렉션 프레임워크
	(collection - 데이터 군 - 자료)
	
	참고) 자바스크립트는 배열은 배열이 아니고 객체, 같은 자료형이 아니더라도 다 포함가능
	순차적인 나열 구조
1) 배열 리터럴로 생성하기
const 배열명 = [] //  new Array()
2)length 프로퍼티
	- 배열 객체의 공간의 갯수
	참고) 
		객체의 속성명이 변수명 규칙과 동일 .(마침표)로 접근가능
		-> 규칙에서 벗어나면 -> ['속성명']
3)Array생성자로 생성하기
const배열명 = new Array(3);
-매개변수가 1개일때는 배열 공간의 갯수
-양의 정수 숫자 가능
new Array(-10) 안됨
-매개변수가 여러개 일때는 갯수만큼 공간이 생성, 값이 순서대로 추가		
6) 배열요소의 추가와 삭제
	-추가 
	push : 끝에 추가
	unshift(..items):앞에 추가
	
	-삭제
	pop() :  가장 뒤의 요소 꺼내기
	shift() : 가장 앞의 요소 꺼내기
	
	-중간 추가, 삭제, 변경
	splice(start,deleteCount,...items(추가할내용))
